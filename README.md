🍔 Dev Burger

O *Dev Burger* é uma interface de hamburgueria funcional desenvolvida para aplicar conceitos avançados de Front-end. O foco principal deste projeto foi a manipulação dinâmica de dados e a criação de uma experiência de usuário (UX) fluida e responsiva.

## 🚀 Tecnologias Utilizadas

* *HTML5*: Estruturação semântica do projeto.
* *CSS3*: Estilização moderna, com foco em responsividade e layout.
* *JavaScript (ES6+)*: Inteligência do projeto, utilizando lógica de programação para renderizar os produtos.

## 🧠 O que eu pratiquei neste projeto?

Este repositório marca o meu progresso no domínio de métodos de array em JavaScript, fundamentais para o desenvolvimento web:

* *map()*: Para percorrer a lista de produtos e exibi-los na tela de forma dinâmica.
* *filter()*: Para criar filtros de categorias (ex: mostrar apenas combos ou apenas bebidas).
* *reduce()*: Para calcular o valor total dos itens, simulando um carrinho de compras ou soma de preços.
* *Manipulação de DOM*: Para atualizar a interface em tempo real conforme as ações do usuário.
